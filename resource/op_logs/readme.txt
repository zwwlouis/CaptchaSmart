prod 线上环境所获得的log文件，其中数据的快速标签通常为‘用户数据’，未快速标签以及打上不确定标签的通常情况下默认为‘用户数据’
---origin 用于训练第一代模型的训练数据，具有最高的可信度，需要确保是‘用户数据’，数据量不需要很大
     ---processed 经过了预处理的数据
---train  用于模型迭代的可信用户数据
     ---processed 经过了预处理的数据
---doubt  可疑的用户数据，对于新增的用户数据，由于不确定是否混有机刷数据，我们通过现有模型对其进行识别，如果数据的识别率过低（比如低于90%），则认为该数据集可疑，进行整体抛弃，并等待人工或其他方式的筛选


test 测试环境所获得的log文件，只提取其中快速标签为‘机器数据’的集合，该标签为专用刷码程序自动添加，可信度较高。忽略测试环境中所有未标签、不确定标签和用户标签的数据。
---train  用户模型训练和迭代的可信机器数据
     ---processed 经过了预处理的数据

extend 拓展数据集，通过其他方式获得的训练数据
